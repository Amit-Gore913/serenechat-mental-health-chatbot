# SereneChat – Mental Health Support Chatbot

## Overview

SereneChat is a simple mental health support chatbot built using React and TypeScript.
The application provides a chat interface where users can share their thoughts and receive supportive responses from the chatbot.

The chatbot also includes a safety filter that detects sensitive self-harm related keywords and provides crisis helpline resources when necessary.

---

## Features

* Chat interface with user and bot messages
* Supportive responses based on user input
* Safety detection for sensitive keywords
* Crisis helpline resource response
* Chat history stored using Local Storage
* Clean and responsive user interface

---

## Technologies Used

* React
* TypeScript
* TailwindCSS
* LocalStorage
* Custom React Hooks
